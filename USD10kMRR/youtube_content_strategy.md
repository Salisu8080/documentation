### Title Formula
`Python [Topic]: [What They'll Learn] (Module N)`

This format hits:
- Primary keyword ("Python") at the start
- Topic specificity for search intent
- Module number for playlist context

### Description Formula (first 2 lines = critical)
1. Line 1: What the video covers + which module — include "Python" + topic
2. Line 2: CTA to pylearn.saypeace.ng with a hook (free, no card)
3. Timestamps for every lesson (boosts YouTube SEO significantly)
4. Hashtags at the very end (8–12 max)

### Tags Strategy
- 3–4 exact-match tags (e.g. "python try except")
- 3–4 broad tags (e.g. "learn python", "python tutorial")
- 2–3 brand tags ("pylearn", "pylearn tutorial")
